# Boutique 2FA&CO

Un site e-commerce moderne et performant permettre aux clients de commander des articles de mode et d'électronique directement via WhatsApp.

## Fonctionnalités

*   **Catalogue Interactif** : Navigation par catégories, recherche instantanée, et pagination.
*   **Panier Dynamique** : Ajout/suppression d'articles, calcul automatique du total.
*   **Commande WhatsApp** : Génération automatique d'un message de commande pré-rempli envoyé directement au vendeur.
*   **Design Premium** : Interface soignée, responsive (mobile-first), et animations fluides.
*   **Galerie Photo** : Vue détaillée des produits avec support multi-images.

## Technologies

*   HTML5
*   CSS3 (Vanilla, Variables CSS)
*   JavaScript (ES6+)
*   Font Awesome (Icônes)
*   Google Fonts (Polices)

## Installation

1.  Clonez ce dépôt.
2.  Ouvrez `index.html` dans votre navigateur.
3.  Pour configurer votre numéro WhatsApp, modifiez la variable `ownerNumber` dans `js/main.js`.
